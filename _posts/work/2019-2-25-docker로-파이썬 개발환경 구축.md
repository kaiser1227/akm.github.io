## 윈도우에서 docker 설치하여 스파크 기반의 jupyter 환경 구축

1. 도커 CE/Toolbox 설치
* http://docker.com

2. 도커 실행하기(로그에 토큰이 출력됨)
> $ docker run -it --rm -p 8888:8888 jupyter/all-spark-notebook
* 위 코드를 실행하면 로그에 토큰값이 출력됨

3. jupyter 접속하기
> http://192.168.99.100:8888/?token=<토큰값>

4. 컨테이너 접속
> $ docker ps –a
> $ docker exec -it "names" bash
